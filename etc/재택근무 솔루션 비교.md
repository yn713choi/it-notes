# 재택근무 솔루션 비교

### 특징
- PC 원격제어 기반의 재택근무 솔루션
- 강력한 보안 정책 적용 (정부기관 납품 이력)
- 실시간 모니터링 및 관리 기능
- Active Directory 연동 지원

### 장점
- 안정적인 원격접속 성능
- 다양한 보안 인증 보유
- 편리한 관리자 콘솔
- 대규모 사용자 지원

### 단점
- 초기 구축 비용이 높은 편
- 복잡한 설정 필요
